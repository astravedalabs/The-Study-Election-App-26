# System Requirements

## Hardware Requirements

### Server Computer

Minimum:

* Intel Core i3 Processor
* 4 GB RAM
* 500 MB Free Storage
* Windows 10 / Windows 11
* Wi-Fi or Ethernet Connection

Recommended:

* Intel Core i5 or better
* 8 GB RAM
* SSD Storage
* Dedicated Election Administrator Device

---

### Voting Stations

Minimum:

* Any Windows Laptop/Desktop
* 2 GB RAM
* Modern Web Browser
* Network Connectivity

Recommended:

* Windows 10/11
* 4 GB RAM
* Google Chrome

---

2. Verify TS Number Database
3. Backup Existing votes.csv
4. Test Admin Dashboard
5. Test Voting Workflow
6. Test Duplicate Vote Protection
7. Start Election
8. Monitor Results
9. Backup Final Results
10. Export Election Data
